## Description
This project is part of my personal portfolio. It presents a 3D room designed using Blender, focusing on modeling interior elements such as walls, floor, and furniture, with applied materials and lighting for a realistic look.

## Project Goal
The main goal of this portfolio project is to experiment with controlling and animating objects, and to explore integrating 3D visual content and animations with a React-based web interface.

## Features
- 3D room modeling using Blender
- Interior objects and furniture design
- Materials, textures, and lighting setup
- Initial experiments with object animation
- Conceptual integration with React for interactive presentation
